# Incendo Board Hardware Wiring Reference
This document serves as the canonical memory for how the physical Incendo boards are wired. 

## Display (TFT)
- `TFT_CS`: 10
- `TFT_RST`: -1
- `TFT_DC`: 8

## LEDs
- `greenLedPin`: 4
- `redLedPin`: 5

## Buttons
- `readButtonPin` (S2): 2
- `writeButtonPin` (S4): 3
- `setButtonPin` (S1): 0
- `wifiButtonPin` (S3): 1
